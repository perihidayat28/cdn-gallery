Gallery gambar dinamis berbasis JavaScript dengan dukungan multi-folder dan filter kategori.

Fitur:
- Load gambar otomatis dari banyak folder
- Filter kategori (All, Sosial Media, E-commerce, Promosi, Keagamaan, Event)
- Lazy loading (lebih cepat & ringan)
- Animasi smooth tanpa kedip
- Tampilan clean & modern

Cocok untuk portfolio, agency, atau showcase konten.
