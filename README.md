🚀 Dynamic Image Gallery (Multi-Folder & Category Filter)

Sebuah sistem gallery gambar berbasis JavaScript yang dirancang untuk menampilkan konten visual secara dinamis, cepat, dan efisien. Project ini mendukung pemuatan gambar dari berbagai folder sekaligus, dilengkapi dengan fitur filter kategori, lazy loading, serta animasi transisi yang halus untuk meningkatkan pengalaman pengguna.

Demo:
Live Preview - [perihidayat.site](http://perihidayat.site/)

✨ Overview

Project ini dibuat untuk kebutuhan portfolio digital, agency kreatif, maupun showcase konten seperti desain grafis, video thumbnail, atau media promosi. Dengan pendekatan berbasis JavaScript murni (tanpa framework berat), sistem ini tetap ringan namun powerful.

Gallery akan secara otomatis membaca dan menampilkan gambar dari berbagai folder, lalu mengelompokkannya ke dalam kategori yang dapat difilter oleh user secara interaktif—tanpa reload halaman.

🔥 Fitur Utama
📁 Multi-Folder Image Loader
Mendukung pemanggilan gambar dari banyak folder sekaligus
Struktur fleksibel untuk kebutuhan skala kecil hingga besar
Mudah dikembangkan atau ditambah kategori baru

- E-commerce
- Promosi

- Perpindahan kategori berjalan smooth tanpa reload
- UX responsif dan cepat

⚡ Lazy Loading Optimization
Gambar dimuat hanya saat dibutuhkan (on viewport)
Mengurangi beban loading awal
Performa tetap ringan meskipun jumlah gambar banyak

🎬 Smooth Animation (No Flicker)
Transisi antar gambar halus
Tidak ada efek “kedip” saat switching kategori
Memberikan kesan profesional & modern

🎨 Clean & Modern UI
Desain minimalis, fokus ke visual konten
Cocok untuk branding personal maupun agency
Responsive (mobile-friendly)
